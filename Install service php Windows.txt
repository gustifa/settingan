Install service PHP di windows
1. buka Command Prompt as an Administrator.
2. Masuk ke direktory nssm/win64/.
3. Defenisikan path file php-cgi.exe. 
4. kemudian pada arguments tambahkan -b 127.0.0.1:9000
5. klik Install Service.
6. klik Win+R kemudian kettikkan services.msc.
7. kemudian service yang sudah dibuat lalu klik start.
